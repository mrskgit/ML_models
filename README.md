README

В репозитории содержаться проекты по подготовке моделей машинного обучения.

Проекты:

Car_prices - на основе исторических данных о продаже автомобилей разработана модель для предсказания цены автомобиля в зависимости от его параметров.

Ore_cleaning - на основе параметров технологического процесса по очистке руды подготовлена модель машинного обучения для предсказания концентрации золота на выходе.

Taxi_orders - на основе исторических данных о заказах такси, с помощью временных рядов подготовлена модель для предсказания количества заказов на следующий час.

Toxic_comments - на основе подборки размеченных комментариев подготовлена модель для определения токсичных комментариев.
