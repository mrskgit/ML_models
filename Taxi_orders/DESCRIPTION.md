## Цель:
Описание проекта
На основе исторические данные о заказах такси в аэропортах нужно спрогнозировать количество заказов такси на следующий час. Необходимо построить модель для такого предсказания.


## Описание данных
Данные лежат в файле taxi.csv. Количество заказов находится в столбце 'num_orders' 

## Используемые библиотеки:

Pandas
Matplotlib
Statsmodels
SkLearn
LightGBM
